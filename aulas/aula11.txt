Exercicio: Desenvolver no Scratch e no NodeJS um APP que informe qual é o combustível mais vantajoso (etanol x gasolina) para abastecer  um carro flex.

o etanol vale a pena quando custar até 70% do valor de gasolina.

//variaveis
etanol, gasolina (númericas)
//entrada
etanol, gasolina 
//processamento/saída
se o preço do litro do etanol custar até 70% do preço de litro da gasolina abastecer com etanol 
senão
abastecer com gasolina

if (etanol < 0.7 * gasolina){
    //abastecer com etanol
} else {
    //abastecer com gasolina
}
